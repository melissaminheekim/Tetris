# 테트리스 게임

다크모드 테마의 클래식 테트리스 게임입니다. HTML5 Canvas를 사용하여 구현되었으며, 부드러운 애니메이션과 직관적인 조작을 제공합니다.

## 🎮 게임 특징

- **다크모드 UI**: 어두운 배경과 네온 블루 테마의 세련된 디자인
- **7가지 테트리스 블록**: I, O, T, S, Z, J, L 블록
- **다음 블록 미리보기**: 다음에 올 블록을 미리 확인 가능
- **라인 클리어 애니메이션**: 라인이 완성되면 깜빡이는 애니메이션 효과
- **레벨 시스템**: 10줄마다 레벨이 올라가며 속도가 빨라짐
- **점수 시스템**: 다양한 점수 계산 방식으로 높은 점수 도전 가능

## 🎯 게임 목표

떨어지는 블록들을 회전하고 이동시켜 가로 한 줄을 완성하세요. 완성된 라인은 사라지고 위의 블록들이 내려옵니다. 블록이 화면 맨 위까지 쌓이면 게임 오버입니다.

## 🎮 조작법

| 키 | 기능 |
|---|---|
| **← →** | 블록 좌우 이동 |
| **↓** | 블록 빠른 낙하 (점수 +1점) |
| **↑** | 블록 회전 |
| **Space** | 하드 드롭 (바로 바닥까지 떨어뜨리기, 떨어진 거리 × 2점) |
| **Enter** | 일시정지/재개 |

## 📊 점수 시스템

### 라인 클리어 점수
- **1줄**: 100점 × 레벨
- **2줄**: 300점 × 레벨
- **3줄**: 500점 × 레벨
- **4줄**: 800점 × 레벨

### 추가 점수
- **빠른 낙하 (↓)**: 1점
- **하드 드롭 (Space)**: 떨어진 거리 × 2점

## 📈 레벨 시스템

- 10줄을 클리어할 때마다 레벨이 1씩 증가합니다
- 레벨이 올라갈수록 블록이 떨어지는 속도가 빨라집니다
- 최소 속도는 100ms까지 감소합니다

## 🎨 블록 종류

1. **I 블록** (청록색) - 4칸 직선
2. **O 블록** (노란색) - 2×2 정사각형
3. **T 블록** (보라색) - T자 모양
4. **S 블록** (초록색) - S자 모양
5. **Z 블록** (빨간색) - Z자 모양
6. **J 블록** (파란색) - J자 모양
7. **L 블록** (주황색) - L자 모양

## 🚀 실행 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. "시작" 버튼을 클릭하거나 Enter 키를 눌러 게임을 시작합니다
3. 방향키와 Space 키를 사용하여 블록을 조작합니다

## 💡 게임 팁

- 다음 블록을 미리 확인하여 전략을 세우세요
- 여러 줄을 동시에 클리어하면 더 많은 점수를 얻을 수 있습니다
- Space 키를 활용하여 빠르게 블록을 배치하세요
- 레벨이 올라갈수록 속도가 빨라지므로 미리 준비하세요
